# Metro Renegade: Complete Python Open-World Game

A fully Python/Pygame interactive open-world action-driving game inspired by modern urban sandbox presentation while remaining original. It is **not a GTA copy**, uses procedural shapes instead of copyrighted assets, and intentionally has **no prone mechanic**.

## Complete Game Features

- Title menu, continue/new game flow, pause state, game over screen, victory screen, and save file support.
- Four-mission campaign: repossession, shooting range, police heat escape, and final delivery finale.
- Third-person-style city exploration with health, armor, cash, score, clock, portrait, wanted stars, mission panels, minimap, and help overlay.
- Seamless on-foot and driving controls with nearby vehicle entry/exit, speedometer, handbrake, garage repair, and multiple vehicles.
- Combat loop with ammo, reloads, hit feedback, target practice, enemy AI, police heat, player damage, armor, and rewards.
- City life simulation with pedestrians, traffic-ready roads, palm-lined streets, buildings, garages, mission markers, particles, and procedural visual polish.
- Progression through cash rewards, score, mission completion, wanted level survival, purchasable armor, and autosave after completed missions.
# Metro Renegade: Python Open-World Prototype

A fully Python/Pygame interactive open-world action-driving prototype inspired by modern urban sandbox games. It is **not a GTA copy** and uses original procedural visuals, systems, and UI.

## Features

- Third-person city exploration with a minimap, cash, clock, portrait HUD, and mission panels.
- Seamless on-foot and driving controls with nearby vehicle entry/exit.
- Simple shooting range gameplay with ammo, targets, hit feedback, and reloads.
- City life simulation: pedestrians, traffic, palm-lined streets, shops, garages, and district labels.
- Interactive missions with objectives, rewards, and distance tracking.
- Vehicle garage with multiple cars and a motorcycle.
- No prone mechanic.

## Install

```bash
python -m pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Controls

| Action | Key |
| --- | --- |
| Start / continue | `Enter` or `Space` on title screen |
| New game | `N` on title screen |
| Move / Drive | `WASD` or arrow keys |
| Sprint / Handbrake | `Left Shift` |
| Enter / Exit vehicle | `E` |
| Interact with mission / garage | `F` |
| Aim | Mouse movement |
| Shoot | Left mouse button or `Space` |
| Reload | `R` |
| Buy armor | `1` |
| Save game | `F5` |
| Cycle radio / district vibe | `Tab` |
| Toggle help | `H` |
| Pause | `Esc` |
| Quit from menu/end screen | `Q` |

## Campaign Goal

Complete every blue mission marker. The game ends in victory after finishing all four missions, including the final delivery mission in the Gold Interceptor. If your health reaches zero, the game ends in a game over state.

## Notes

This project intentionally avoids copyrighted game assets and trademarked names. It delivers a playable, Python-only urban sandbox game with original art, mechanics, and GUI presentation.
